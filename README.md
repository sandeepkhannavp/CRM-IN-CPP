# CRM-in-c++
Extension of an exesting CRM - for (20XT25) OOPS LAB @PSG COLLEGE OF TECHNOLOGY


A CRM project written in C++ where companies can

- Manage customers
- Manage Leads
- Manage Feedbacks
- Track customer cycle
- Store their data and pull theirata whenever needed
