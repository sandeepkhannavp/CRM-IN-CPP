# CRM-in-c++
Written on top of an existing CRM - for (20XT25) OOPS LAB @PSG COLLEGE OF TECHNOLOGY


A CRM project written in C++ where companies can

- Manage customers
- Manage Leads
- Manage Feedbacks
- Track customer cycle
- Store their data and pull theirata whenever needed
